## Weather Forecasting App

A mobile-friendly weather application that fetches real-time weather information using the OpenWeatherMap API. The app uses asynchronous JavaScript operations to display live temperature, weather conditions, and location-based forecasts.

### Features
- Real-time weather updates
- Search weather by city
- Dynamic weather display
- Responsive UI design
- User preference storage using Local Storage

### Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- OpenWeatherMap API
- Fetch API
- Async/Await
- DOM Manipulation
- Local Storage
